%u 는 10진수를 출력하는 용도 ( 결국 양수)
만약에 -1 출력시 부호가 있으니 전혀 다른 결과가 나옴.

%f 
float a=9;
-> 9.00000

%o : 8진수

%x : 16진수

%u : unsinged int 부호없는 10진수 (음수 안됨)

%p : 포인터의 주소

%g : 실수형 자동출력

long형 범위 -2148579123 ~ +2148579123
int 표현범위 20억

float 이 실수형 double 이 좀더 소숫점 길게 나타낼수 있음.

scanf시 ("%2d",a);
입력 : 1 -> ok   12 -> ok    123 -> 12만저장.

\a 알람
\t 탭
\r 커서 맨앞으로,

%g 는 3.1400000 뒤에 0같은거 깔끔하게 정리.
